# Internet Programming — Coursework (University of Western Macedonia)

Assignment solutions for the *Programming on the Internet* course. The repository is kept here primarily to demonstrate version control habits, project organization, and the ability to work across multiple languages in an academic context.

> **Note:** JavaScript is not my primary language. These are coursework exercises, not production-level JS.

---

## Assignments

| File | Description |
|---|---|
| `solution_24.js` | `filterRange` — returns array elements within a range (non-mutating) |
| `solution_25.js` | `filterRangeInPlace` — filters array values in place |
| `solution_26.js` | `mySort` — sorts numbers in descending order |
| `solution_27.js` | `copySorted` — returns a sorted copy of an array of strings |
| `solution_28.js` | `getNames` — extracts name properties from object arrays |
| `solution_29.js` | `Calculator` — extendable calculator with custom operators |
| `solution_30.js` | `fullNames` — maps users to include `fullName` + `id` |
| `solution_31.js` | `sortByAge` — sorts objects by age |
| `solution_32.js` | `getAverageAge` — computes average age from objects |
| `solution_33.js` | `unique` — removes duplicate elements |
| `solution_34.js` | `groupById` — groups objects by id into a lookup object |
